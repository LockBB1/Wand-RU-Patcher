# Как помочь проекту

## Добавить перевод игры (самый ценный вклад)

Точные словари лежат в `WAND_RU/renderer/games/<gameId>.json` - по файлу на игру:

```json
{
  "title": "Название игры",
  "names": {
    "Unlimited Health": "Бесконечное здоровье",
    "Set Prestige": "Задать престиж"
  }
}
```

- `gameId` - числовой ID игры в Wand (виден в запросе `api.wemod.com/v3/games/<id>/trainer`).
- Ключи `names` - **точные** английские имена читов из трейнера (регистр и символы важны).
- Значения - русский перевод. Имена собственные и аббревиатуры (HP, XP, UV) можно оставлять.
- Не знаете gameId - создайте [запрос игры](../../issues/new?template=game-request.yml), добавим сами.

После правки словаря выполните `node WAND_RU/renderer/build-hook.mjs` (перегенерирует `cheat-hook.js`)
и `node --test WAND_RU/renderer/*.test.mjs` - все тесты должны быть зелёными. Оба файла кладите в PR.

## Сообщить о проблеме

- [Запрос игры](../../issues/new?template=game-request.yml) - хотите 100% перевод читов вашей игры.
- [Баг](../../issues/new?template=translation-bug.yml) - кривой перевод или ошибка установщика.
  Приложите лог: в установщике «Показывать лог» -> «Экспорт» -> `.md` файл.

## Правила кода

- Только короткие дефисы «-», длинные тире не используем.
- Новая логика - новый тест (`node:test` для renderer, xUnit для C#).
- Сборка: `WAND_RU/build.bat` (`-Test` для прогонов, `-Publish` для exe).
