# Smoke-чеклист — Wand-RU-Installer

Ручная проверка перед релизом. **Гоняет пользователь** (не Claude). Wand запускается только здесь.

## Сборка
- [ ] `WAND_RU\build.bat -Test` — build + все тесты зелёные.
- [ ] `WAND_RU\build.bat -Publish` — создан `WAND_RU\publish\WandRuInstaller.exe` (single-file, ~130 МБ).

## Установщик (запуск exe)
- [ ] `WandRuInstaller.exe` стартует, окно рисуется (безрамочное, тень, перетаскивается за шапку).
- [ ] Версия в шапке совпадает с текущей (`v 0.4.x`).
- [ ] Wand найден автоматически → экран **Ready** («Wand 12.37.0» + кнопка «Русифицировать»).
  - [ ] Если не найден → **NotFound** + «Обзор…» открывает выбор папки; указать корень Wand → **Ready**.
- [ ] Шестерёнка → **настройки**: выпадающий список версий app-*, чекбоксы «Перезапустить Wand», «Показывать лог». «Закрыть» скрывает попап.

## Патч (на реальном Wand)
- [ ] «Русифицировать» → **Working** (индикатор + лог) → **Done** («Готово! Перезапустите Wand → Настройки → Язык → Русский»).
- [ ] Запустить **Wand вручную** → Настройки → Язык → **Русский** есть в списке (с флагом РФ) → выбрать → UI на русском.
- [ ] Строки переведены (шапка, меню, настройки) — сверить с `ru-overrides.json`.

## Откат
- [ ] Перезапустить установщик → состояние **Patched** → «Откатить» → возвращается в **Ready**.
- [ ] Wand снова на английском (или языке по умолчанию), `ru-RU` из списка исчез.

## Локализация UI установщика
- [ ] Система на русском → UI установщика на русском.
- [ ] Система на английском → UI установщика на английском.

## Крайние случаи
- [ ] Повторный «Русифицировать» поверх пропатченного — не ломает (идемпотентно).
- [ ] Настройка «Перезапустить Wand после установки» — Wand стартует сам после патча.
