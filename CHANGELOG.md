# Что нового

Заметки к релизам. Секция с номером версии попадает в тело GitHub Release автоматически
(тег `X.Y.Z` -> workflow `Release` берёт отсюда текст). Нет секции - в релиз уедет список коммитов.

## 0.17.7

Релиз про честность: русификатор больше не рапортует успех там, где не проверил результат.

- **Откат больше не умирает молча.** Если бэкап оригинала снесли (антивирус, чистильщик, руки),
  а Wand уже русифицирован, WRP раньше сохранял *пропатченный* `app.asar` как «оригинал» - откат
  навсегда переставал возвращать чистый Wand. Теперь такой случай виден: WRP предупреждает и
  спрашивает, продолжать ли без возможности отката.
- **Откат проверяет, что запись прошла.** После отката WRP перечитывает контрольную сумму из
  `Wand.exe`: раньше при неудачной записи он говорил «Откат завершён», а Wand молча не запускался.
- **Честный итог вместо безусловного «Готово».** На экране после установки видно, что реально
  легло: `локаль ✓ · читы ✓ · карты ✓`. Если якорь какого-то куска не нашёлся на вашей версии
  Wand - это написано прямо, а не спрятано.
